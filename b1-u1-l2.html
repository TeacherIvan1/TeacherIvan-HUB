2. Grammar: The Unfinished Bridge

In Level A2, you learned that we use the Past Simple when a time period is completely FINISHED (like yesterday). But what happens if the time period is still happening?

We use the Present Perfect. It acts like a "bridge" connecting a past action to an unfinished present time!

Past Simple

The time is 100% finished. The bridge is broken.

Yesterday
Last Week
In 2025

"I ran 5k yesterday."

Present Perfect

The time is NOT finished. The bridge connects to NOW.

Today
This Week
This Morning

"I have run 5k today."

🌉 WHY "TODAY"?

If you say "I have run 5k today", it means today is not over. You might run more later! The action is completed, but the time period (today) is still open.
